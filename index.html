FAQ

Arwin is a novel architecture based on the Ph.D. thesis of our founder (Joaquín) and its code was built from zero, so it's not a fork nor a platform over another crypto.

Arwin's first testnet was launched at May, 2019. At January 3rd of 2020 the mainnet was launched.

Arwinsh (ARW) is the native unit of account of the Hathor Network, created as a utility token needed to access the decentralized payment, contracting and token issuance capabilities of our technologies.

Arwin achieves consensus using proof-of-work and the same hash of bitcoin (sha256d).

If you want to mine Arwin (ARWs), first you need to install one of the available mining software, then you can connect to a full-node to start mining. Ideally, you should run you own full-node for security reasons. See more information at our mining page.
